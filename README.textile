*Bookland* provides the ISBN class whose instance represents an ISBN code. The class abstracts ISBN-10 and ISBN-13s and provides methods to one to the other.

    $ irb -r "bookland"
    >> book = ISBN.new("0262011530")
    => "0262011530"
    >> book.isbn13
    => "9780262011532"
    >> book.to_s(1,3,5)
    => "0-262-01153-0"
    >> ISBN.new("9780262011532") == book
    => true
    >> ISBN.new("0262011531")
    => false
